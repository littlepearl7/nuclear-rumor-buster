# 核规联盟 · 科普互动平台

一个模拟微信群聊的核安全科普辟谣互动网页，通过角色扮演+答题闯关的方式，向中老年群体普及辐照食品、核电站、CT检查、食盐安全等领域的科学知识与法律法规。

## 快速开始

直接浏览器打开 `index.html` 即可运行，无需任何构建工具或后端服务。

## 如何使用

1. **启动页面** — 打开后进入原子动画开场，请在下方选择要挑战的谣言（或选择随机分配），点击「确认并进入系统」开始。

2. **阅读谣言** — 群聊中会自动弹出"大爷大妈"们发送的谣言消息，仔细阅读他们担心的具体内容。

3. **点击主按钮答题** — 底部中央的大按钮会显示当前步骤名称（如「第一步：查官方通报」），点击后弹出三个选项。

4. **选择正确选项** — 三个选项中只有一个是专业、硬核的辟谣回复。选错会显示错误原因解析，并可点击「重选本题」重新选择。

5. **推进流程** — 答对后，小原会自动发送辟谣内容，并推进到下一步。每个谣言均按以下顺序进行：
   - 第一步：查官方通报（展示权威数据来源）
   - 第二步：讲科学原理（解释核心科学机制）
   - 第三步：亮法规支撑（引用具体法律条文）
   - 第四步：我们该怎么做（给出行动指南）

6. **查阅法规** — 桌面端右侧面板会实时收录辟谣过程中引用的法律条文，点击链接可查看原文。

7. **完成辟谣** — 四步全部完成后，群友会发送正面反馈，并获得一枚徽章。点击「新挑战」可继续破解下一个谣言。

8. **全部完成** — 破解所有谣言后，会弹出成就证书，可下载保存。

### 快捷操作

- **一键完整辟谣** — 底部「⚡ 一键完整辟谣」按钮可跳过答题，直接展示全部四步辟谣内容（适合快速预览）
- **重新挑战** — 选错导致科普失败时，点击弹窗中的「重新挑战」可重置当前谣言
